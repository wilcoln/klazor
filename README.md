# klazor
Klazor is a binder...
## Installation
```bash
git clone git@github.com:wilcoln/klazor.git
cd klazor
npm install
env/bin/pip install -r requirements.txt
```

## Setup
```bash
env/bin/python manage.py migrate
env/bin/python manage.py createsuperuser
```
### Run
```bash
env/bin/python manage.py runserver
```
